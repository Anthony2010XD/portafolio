Sobre estas categorías

Selecciona una categoría arriba para ver una breve descripción y ejemplos.

Cómo funciona

  1. Los botones tienen el atributo data-description con el texto que se mostrará al hacer clic.
  2. El pequeño script al final añade controladores de clic a cada botón y actualiza el área de descripción.
  3. Para añadir una categoría: copia un botón, cambia su etiqueta y su data-description.
